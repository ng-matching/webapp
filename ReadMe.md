## Features:
  * Arcade Mode
       - Easy (12 cards)
       - Intermediate (20 cards)
       - Hard (28 cards)
  * Time Attack
       - Easy (12 cards)
       - Intermediate (20 cards)
       - Hard (28 cards)
  * Color Scheme selection
  * Restart, Home ,Quit, guesses, tries
